# Natural Language Processing

This repository documents my study of **Natural Language Processing (NLP)** through a collection of Jupyter notebooks containing concept explanations, implementations, and experiments.

The objective of this repository is to build a strong foundation in NLP by exploring both fundamental and advanced topics while applying them through practical coding exercises. Each notebook focuses on a specific concept and includes relevant theory, implementation details, and observations.

## Topics Covered

The repository includes notebooks on:

- Text Preprocessing
- Tokenization
- Stopword Removal
- Stemming and Lemmatization
- Regular Expressions

Additional topics and projects will be added as the learning process progresses.

## Tech Stack

- Python
- Jupyter Notebook
- NumPy
- Pandas
- NLTK
- Scikit-learn

## Purpose

This repository serves as:

- A record of my NLP studies
- A collection of practical implementations
- A reference for future projects and research
- A demonstration of my technical growth in Artificial Intelligence and Machine Learning

## Note

The notebooks are organized according to the sequence in which topics were studied. As my understanding of NLP evolves, existing notebooks may be updated with improved implementations and additional insights.
